企业介绍

      中建担保有限公司（以下简称中建担保），成立于2004年，立足深圳,辐射全国,以增强企业信用,降低企业经营风险为己任，是一家为企业提供商业担保及相关金融产品服务的全国性专业担保公司。随着中建担保业务蓬勃发展，2008年12月于金融危机中逆势而上，在美国纳斯达克OTCBB市场成功上市（股票代码：SNAS.OB）。
      中建担保总部位于深圳市福田CBD的新世界商务中心，并在北京、广州、深圳、天津、大连、成都、重庆、长沙、武汉、石家庄、太原、西安、杭州、南京、苏州、昆明、贵阳、济南、南宁、海口、郑州、中山、东莞等近30个城市设立了分支机构，拥有全国最大的企业担保业务服务平台，业务规模荣居同行业前!
      中建担保自成立七年来，先后与中国建设银行、中国工商银行、中国银行、交通银行、上海浦东发展银行、深圳平安银行等大型国有及商业银行形成了长期战略合作伙伴关系，并为中国建筑集团、中铁建工、万科集团、保利集团、沃尔玛、美的集团等大型企业提供了全方位的担保服务，同时凭借自身的专业和实力承保了中央电视台新台址建设工程、国家博物馆改扩建工程、北京国际汽车博览中心、深圳大运会主会场、京沪高铁工程、武广高铁等一系列国家名优重点工程项目。
      中建担保通过锲而不舍的努力，先后在行业内获得了“全国信用担保十佳诚信单位”、“全国共铸诚信宣传示范单位”、 “中国信用担保机构最佳创新奖”、“中国100家最具发展潜力品牌创新企业” 及“中国知名品牌最佳创新奖”等荣誉称号，更是成为北京市、深圳市等各地担保协会的主要理事单位。 展望未来，中建担保仍将继续弘扬开放、务实、进取、和谐的企业精神，不断推动全社会诚信建设的完善，坚持“共建双赢、伴您成长”的服务理念，与我们的客户和合作伙伴们一起，共同为中国各行各业的发展壮大和国家建设做出应有的贡献！
